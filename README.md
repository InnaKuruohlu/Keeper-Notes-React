A simple Note-Taking App built with HTML5, CSS, JavaScript and React.js.

Allows users to create, view and delete notes dynamically.


Features: 

- Add new notes 
- Delete notes
- Responsive layout
- Styled with CSS and Material UI icons



Technologies Used:

- React 19
- React DOM
- Material UI
- CSS
- gh-pages gor GitHub Pages deployment 


Check out the Live Version: https://innakuruohlu.github.io/Keeper-Notes-React/
